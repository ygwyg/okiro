# ASCII Animation Package

This package contains your converted ASCII animation.

## Files:
- ascii-frames.json: Complete animation data (recommended)
- ascii-metadata.json: Animation metadata
- ascii-frame-XXX.json: Individual frame files

## Usage:
See the example code in the converter tool for implementation details.

## Settings Used:
- Resolution: 4
- Color Mode: color
- Brightness: 1
- Contrast: 1
- ASCII Characters: " .,:;i1tfLCG08@"
- Frame Delay: 400ms
- Total Frames: 122
